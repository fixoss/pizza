# Pizza!

The most over-engineered pizza calculator ever!

## Usage:

Run

```
$ python3 pizza.py
```

Enter your inputs to the recipe:
 * number of dough balls
 * ball weight
 * hydration amount
 * etc...

It will print out the appropriate measurements to make the best tasting pizza pie.
